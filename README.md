# Fullstack real-time chat application
Using MERN stack
IOS notification sound built-in
Real time online status
UI renders with glassmorphism effects using @TailwindCSS and @DaisyUI

# Authentication
Built-in authentication system using @JSONwebtokens
Uses middlewares and protected routes for authentication
Built-in API used for error handling and displays error messages

# Clean code base
Services separated into their own folders

# Database
Using @MongoDB to store (Users, Passwords, Conversations)
Models used to define structure of this data

# Frontend
Using @React 
Custom hooks designed to carry out operations such as (Sign in, Log out, and messaging users)

# State management solutions
Using @ReactContext and @Zustand in frontend for any changes such as getting the users id and storing/sending or getting messages.

# Realtime communication
@Socket.IO allows data to be sent to server and client in a bi-directional manner
Intergrated into @Express server

# Deployment

https://chat-app-1m9x.onrender.com/
